# Minimalist CV

Simple web app that renders minimalist CV with print-friendly layout.

Built with Next.js and shadcn/ui, deployed on GitHub Pages.

# Run

```bash
yarn install
yarn dev
```

# Build

```bash
yarn build
node scripts/fix-og-images.mjs
```
